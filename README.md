# 📖 Shopi <a name="about-project"></a>

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Node js

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/Felipe-Perez-Ferraro/shopi
```

### Usage

## Front-End
```sh
  cd frontend
  npm install
  npm run dev
```

## Back-End
```sh
  cd backend
  npm install
  npm run dev
```
